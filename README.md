CodeTrack 
A full-stack web application to track and organize your coding problem-solving journey. Add problems, group them into sets, and never lose track of what you've solved.
Features

Add and manage coding problems
Organize problems into custom sets
Track your solving status
User authentication & profiles

Tech Stack

Frontend: React + Vite
Backend: Node.js + Express
Database: MongoDB

Environment Variables
Rename .env.example to .env in both frontend and backend folders and fill in your credentials.
